# Испанский в диалогах

Интерактивный курс испанского языка для самостоятельного обучения — 55 диалогов уровня A2–C1 с аудио, упражнениями и контролем прогресса.

**Разработчик:** Громова Анна — преподаватель, методист, переводчик, основатель ИНТЕРАКТИВНОЙ ШКОЛЫ МУДРО (зарегистрированный товарный знак).

## Возможности

- **7 шагов обучения** — от подготовки до закрепления материала
- **Озвучивание** — вся лексика, диалоги и грамматика озвучены (Web Speech API)
- **Проверка произношения** — Shadowing и Roleplay с распознаванием речи (Chrome/Edge)
- **Игры и упражнения** — Memory Match, Fill-Gap, Quiz, диалог по ролям
- **Прогресс** — результат сохраняется в браузере (localStorage)
- **Адаптация под мобильные** — корректно отображается на телефонах и планшетах

## Структура курса

Курс состоит из 6 частей по тематическим блокам:

| Часть | Уровень | Темы |
|---|---|---|
| 1. ¡Hola! ¿Qué tal? | A2 | Приветствия, знакомство |
| 2. Mi mundo | A2–B1 | Семья, дом, описание |
| 3. Día a día | B1 | Повседневная жизнь |
| 4. Viajes y ocio | B1–B2 | Путешествия, досуг |
| 5. Trabajo y estudio | B2–C1 | Работа, учёба |
| 6. Temas de actualidad | C1 | Актуальные темы, дискуссии |

Каждый диалог содержит: лексику, грамматику, диалог с построчным переводом, разговорную практику, тест и домашнее задание.

## Как пользоваться

1. Откройте [главную страницу](https://mudrois.github.io/espanol-en-dialogos/)
2. Выберите часть курса, затем диалог
3. Проходите шаги последовательно (подготовка → лексика → грамматика → диалог → говорение → тест → рефлексия)

## Методические рекомендации

### Для кого этот курс

Курс предназначен для взрослых и подростков (14+) с начальным (A2) и средним (B1–C1) уровнем испанского. Подходит как для самостоятельного изучения, так и для работы с преподавателем.

### Как построен курс

Каждый диалог — это законченный урок, построенный по принципу «от простого к сложному» и охватывающий все языковые аспекты:

| Шаг | Что делать | Время |
|---|---|---|
| 1. Подготовка | Прочитайте цели урока, настройтесь на тему | 2–3 мин |
| 2. Лексика | Изучите слова, прослушайте произношение, сыграйте в Memory Match | 10–15 мин |
| 3. Грамматика | Разберите правило, выполните Fill-Gap | 10–15 мин |
| 4. Диалог | Прослушайте диалог, прочитайте с переводом, потренируйте Shadowing | 15–20 мин |
| 5. Говорение | Составьте свой диалог, сыграйте Roleplay | 10–15 мин |
| 6. Тест | Проверьте себя | 5–10 мин |
| 7. Рефлексия | Закрепите результат, запишите домашнее задание | 3–5 мин |

**Рекомендуемое время на один диалог:** 45–60 минут.

### Методические принципы

1. **Аудирование и повторение.** Каждую новую фразу важно не только прочитать, но и услышать и повторить вслух. Используйте кнопку 🔊 для прослушивания, а затем функцию Shadowing (повторение с микрофоном).

2. **Контекстное изучение.** Лексика и грамматика даются в контексте живого диалога, а не изолированно. Это помогает запоминать не отдельные слова, а готовые речевые обороты.

3. **Принцип одного правила.** На каждый диалог — одно новое грамматическое правило. Не пытайтесь освоить всю грамматику сразу: двигайтесь последовательно.

4. **Продуктивная речь.** После пассивного усвоения (чтение, аудирование) обязательно переходите к активной практике: составьте собственный диалог, пройдите Roleplay.

5. **Регулярность.** Занимайтесь по 1 диалогу в день, а не по 5 диалогов раз в неделю. Регулярные короткие занятия эффективнее редких длинных.

6. **Цикличность.** После прохождения 5–6 диалогов возвращайтесь к первым — вы заметите, как выросли. Повторение — ключ к долговременной памяти.

### Советы для самостоятельной работы

- **Ведите словарь.** Записывайте новые фразы в тетрадь или приложение (Anki, Quizlet). Указывайте контекст — целую фразу, а не одно слово.
- **Читайте вслух.** Даже без функции распознавания речи чтение вслух активирует моторную память и улучшает произношение.
- **Не бойтесь ошибок.** Ошибки в тесте и упражнениях — часть процесса. Анализируйте их и возвращайтесь к сложным темам.
- **Используйте дополнительный контент.** Смотрите видео на YouTube на испанском, слушайте подкасты (Radio Ambulante, Españolistos), читайте адаптированные тексты.
- **Практикуйте язык в жизни.** Меняйте язык на телефоне на испанский, пишите короткие заметки о своём дне.

### Для преподавателей

Курс можно использовать как дополнительный материал к основному учебнику или как основу для разговорного клуба:

- **На занятии:** пройдите шаги 1–4 вместе, шаги 5–6 дайте как домашнее задание.
- **В разговорном клубе:** используйте диалог как основу для обсуждения, Roleplay — для ролевых игр в парах.
- **Контроль:** прогресс ученика сохраняется в браузере — можно проверять результаты тестов в начале каждого занятия.

## Технические требования

- Современный браузер (Chrome, Edge, Firefox, Safari) — для базовой функциональности
- **Chrome или Edge** — для распознавания речи (Shadowing, Roleplay)
- JavaScript включён
- Наличие микрофона — для Speaking-упражнений

> **Примечание:** Распознавание речи (SpeechRecognition) работает только в браузерах на основе Chromium. В Firefox и Safari упражнения на говорение доступны в режиме самопроверки.

## Установка и запуск локально

Курс работает как статический сайт. Сервер не требуется.

```bash
git clone https://github.com/MUDROis/espanol-en-dialogos.git
cd espanol-en-dialogos
# Просто откройте index.html в браузере
# или запустите локальный сервер:
python -m http.server 8080
```
